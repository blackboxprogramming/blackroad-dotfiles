# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Reporting a Vulnerability

This is proprietary software owned by BlackRoad OS, Inc.

To report a security vulnerability, please contact:
- Email: security@blackroad.os (preferred)
- Do NOT open a public issue for security vulnerabilities

We will acknowledge receipt within 48 hours and provide a detailed response
within 5 business days.

## Security Practices

- All GitHub Actions are pinned to specific commit SHAs
- Dependabot is enabled for automated dependency updates
- CodeQL analysis runs on every push and pull request
- Dependency review is required for all pull requests
- Branch protection is enabled on `main`
